# PortWatch
PortWatch: panel web local para macOS que muestra conexiones salientes por aplicación y prioriza el riesgo. Modo simple con frases claras, cola de revisiones y vista técnica (score 0–10, firma Apple/tercero, beaconing). Usa psutil con fallback a lsof. Datos locales en el navegador. Python 3.10+. Detección de servidor local.
